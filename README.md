# SmartBudget – Login Responsive

Este proyecto corresponde a un trabajo práctico, donde se desarrolla una pantalla de inicio de sesión (login) a partir de un diseño simple realizado en Figma.

El objetivo fue aplicar lo aprendido sobre estructura HTML, estilos CSS, uso de SASS y Bootstrap, de forma ordenada y progresiva.

## Descripción
La aplicación muestra una pantalla de login que incluye:
- Una barra de navegación (navbar)
- Un formulario central con campos de email y contraseña
- Un botón de ingreso
- Un pie de página (footer)

El diseño es responsive, por lo que se adapta a distintos tamaños de pantalla como móvil, tablet y escritorio.

## Tecnologías utilizadas
- HTML5  
- CSS3  
- Metodología BEM para nombrar clases  
- SASS para organizar los estilos  
- Bootstrap 4 para componentes y utilidades  
- Git y GitHub para control de versiones  

## Contenidos aplicados
Durante el desarrollo del proyecto se trabajaron los siguientes temas:

- Diseño previo usando un prototipo en Figma
- Estructura HTML semántica
- Organización de estilos con BEM
- Modularización de CSS usando SASS y estructura 7-1
- Modelo de cajas, Flexbox y diseño responsive
- Integración y personalización de Bootstrap 4

## Diseño Responsive
El proyecto fue probado en diferentes tamaños de pantalla:
- Móvil (375px)
- Tablet (768px)
- Escritorio (1280px)

Las capturas de estas pruebas se encuentran en la carpeta **/evidencias**.

## Estructura del proyecto
El proyecto está organizado en carpetas para mantener el código ordenado y fácil de entender, separando estilos base, componentes y layouts.
